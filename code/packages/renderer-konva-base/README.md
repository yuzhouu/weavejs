# Weave.js / Renderer Simple

This package generates the `@inditextech/weave-renderer-simple` package, the Weave Renderer Simple provides a renderer
using Konva and simplified and performant reconciler.

## Setup

This is a monorepo, to install this package dependencies, just setup the monorepo, this can be done by locating on the `/code` project and execute the following command.

```sh
$ npm install
```

## Usage

This is a monorepo, this commands need to be from the `/code` folder of the repo.

### Build the package

```sh
$ npm run build --workspace=@inditextech/weave-renderer-simple
```

### Link the package

```sh
$ npm run link --workspace=@inditextech/weave-renderer-simple
```

### Lint the package

```sh
$ npm run lint --workspace=@inditextech/weave-renderer-simple
```

### Test the package

```sh
$ npm run test --workspace=@inditextech/weave-renderer-simple
```
